Features

DroidPHP comes with following features

Nginx v1.5.1

lighttpd v1.4.34

PHP v5.5.9

MySQL v5.1.62

MSMTP v1.4.3

Usefull options

+ possibility to run the server on port:80; (root required)

+ possibilit to use lighttpd or nginx
